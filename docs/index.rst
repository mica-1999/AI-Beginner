AI Beginner Documentation
=========================

Welcome to the AI Beginner project documentation! This project involves a Tic-Tac-Toe game AI that learns through experience using Reinforcement Learning (RL). The goal of this AI is to learn optimal strategies by playing games and refining its decisions over time. The project is written in Python and is designed to help beginners understand the basics of AI and reinforcement learning.

Installation
============
To set up this project on your local machine, follow the steps below:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/yourusername/AI-Beginner.git
   cd AI-Beginner

2. **Set up the virtual environment**:
   To create a virtual environment, run:
   ```bash
   python -m venv venv
   Activate the virtual environment: On Windows, run:
   .\venv\Scripts\Activate
   If you encounter an issue with scripts being disabled, use the following command in PowerShell:
   Set-ExecutionPolicy RemoteSigned -Scope CurrentUser
   Install dependencies: After activating the virtual environment, install the required Python libraries using
   pip install -r requirements.txt

Usage
To run the Tic-Tac-Toe AI, activate the virtual environment and start the game script. The AI will begin learning through its own experiences in playing the game.
